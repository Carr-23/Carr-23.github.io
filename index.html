Projects

        Breaker Duel: This is a 2 player versus game inspired by the games Pong and Breakout. I was the head developer on this project since I had more knowledge with Unity Engine. I was in charge of the majority of the programming and creating all of the assets of the game. After completion, we published Breaker Duel on the Google Play Store.
        Konflyx.io: This is a single-player conflict resolution game.This game teaches the player how to handle conflict in different stages in their life, kid, adolescent, and adulthood. Using Object-Oriented Programming I created classes for everything that was gonna be repeated, such as letters. This allowed us to put together a final version of the game fairly quickly. This also allows for external modifications to be made. People have edited the text file to make a new script and make a different story, and thanks to OOP this can be done without any further code required.
        iCar: Created an "Intelligent Car" that would be able to follow a black line on its own without assistance. I was the project manager and looked over the programming section. On the other hand, I did most of the hardware work, since it was my strong suit. The iCar was a school project and everyone had to create on, so we took it upon ourselves to make the best one. Our car was the only one that was able to complete the final level, not only that, but our car was the fastest of the bunch.
